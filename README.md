## 🚀 About the Project
The **Blood Bank Management System** is designed to:
- Manage donor records
- Track blood availability
- Handle blood requests efficiently
- Provide insights and search functionality

This system helps improve **blood donation coordination** and **emergency response**.

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/Blood_Bank_Management_system.git
```
### 2. Navigate to project
```bash
cd Blood_Bank_Management_system
```
### 3. Install dependencies
Backend:
```bash
pip install -r requirements.txt
```
Frontend:
```bash
npm install
```
### 4. Run the project
Backend:
```bash
python main.py
```
Frontend:
```bash
npm start
```
